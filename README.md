# Proof Pages (Divi 5) — Live Demo

This repository hosts a **rendered static export** of the Proof Pages layout pack
for Divi 5, published for Divi Marketplace review. Every family and page is
browsable with no login.

- **Family index:** [`/`](./index.html)
- **12 design families**, each with 8 pages: Home, Case Studies, a featured case
  study, Harbor & Pine, Portfolio, Client Words, Start a Review, and a Sections
  showcase — 97 pages total.

This is a static, read-only preview of the front-end only. The importable Divi
layout product (JSON exports) is **not** included here.
